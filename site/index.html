Monetary policy

Scarcity that stays readable.

SRCE is issued only through mining rewards. The first reward era creates about half of the supply, then every era cuts the reward in half. The supply cap is enforced by node consensus rules.

Mining

Every coin starts as work.

Miners build blocks, search for valid Proof-of-Work, and receive SRCE rewards when the network accepts their blocks.

Wallet

Keys create addresses. Nodes verify coins.

A Saracie wallet derives addresses from a mnemonic seed. Coins live on the blockchain as UTXOs, and the wallet controls the keys that can spend them.
