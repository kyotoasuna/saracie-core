# Saracie Core v0.1.2

This release makes the public zip packages easier to use.

## Changes

- Release packages now include startup scripts and user docs.
- PowerShell scripts work from both the source repository and extracted release zips.
- Latest release link updated to v0.1.2.

## Included

- `saracied`
- `saracie-miner`
- `saracie-wallet`
- `saracie-ui`
- `scripts/`
- quickstart, mining, wallet, UI, security, whitepaper, and license docs
