Saracie Core v0.1.0

Initial public foundation release for Saracie (SRCE).

Includes:

- Saracie consensus parameters
- 210,000 SRCE maximum supply
- 60-second block target
- six-month halving schedule
- difficulty retarget every 60 blocks
- genesis block
- Proof-of-Work mining
- signed transactions
- UTXO validation
- local mempool
- peer sync and gossip
- encrypted wallet files
- local browser UI
- Windows/Linux binaries

```text
Premine: 0 SRCE
Initial reward: 0.39954337 SRCE
Halving interval: 262,800 blocks
```
